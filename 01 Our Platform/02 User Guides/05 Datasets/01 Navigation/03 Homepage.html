-Each dataset has an
   - Introduction: High-level overview of what the dataset includes/covers, who it's created by, and how it's created/gathered
   - About the provider: Description of the data provider, with links to their website
   - Getting Started: A few lines of code, everything you need to add the dataset to your algorithm
   - Data Summary: A table of start date, coverage, resolution, density, timezone
   - Example applications for using the dataset in your algorithms
   - Data point attributes: All the classes + properties you can use from the dataset
   - Price: Price of accessing the dataset
         - Cloud access - Using in the cloud, sometimes free, sometimes have to pay with dollars on your monthly bill
         - Download - Using locally, can generate the CLI command with the generator, have to pay with QCC
                          - Automatically formatted correctly for you in your local machine
