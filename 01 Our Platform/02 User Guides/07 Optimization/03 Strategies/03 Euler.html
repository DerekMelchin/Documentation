What is Euler search:
-Multiple grid searches at different granularities
-(1) Selects values for each parameter.
-(2) Calculates the value of the objective function
-(3) The parameter values that minimize/maximize the objective function are selected and another grid search surrounding those values (at a finer granularity) are selected & tested.
-Steps 1-3 are repeated until the optimizer reaches a granularity limit for each parameter

Why use Eluer search (pros):
-More comprehensive search than grid search

Why not to use Eluer search (cons):
-Takes longer to execute when compared to grid search
-When scanning at a large granularity, the optimizer can skip over optimal parameter values
