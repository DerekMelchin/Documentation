You need to select a node to deploy a live algorithm. To view the live trading nodes in your organization, call the ReadNodes method with your organization Id.

var nodes = api.ReadNodes(organizationId);
nodes = api.ReadNodes(organization_id)

To get the organization Id, open the organization homepage. The organization Id is in the URL. For example, the organization ID of https://www.quantconnect.com/organization/d6d62db48592c72e67b534553413b691 is d6d62db48592c72e67b534553413b691.
